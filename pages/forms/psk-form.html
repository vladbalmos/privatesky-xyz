Psk-form component is a wrapper over psk-container, so the functionalities available in psk-container are also in psk-form.

The purpose of the component is to create an user form.

The form submition will be handled by the controller attached via controller-name attribute. Also, the model for the form will be defined in the same controller.

At the bottom of the page stands a minimal example of using the component with its attributes. More examples can be found within Using forms page.

As HTML children, any component can be rendered inside.
